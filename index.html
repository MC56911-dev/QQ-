Detailed Style Introduction

Each style supports the same workflow: set the scene first, then sculpt the typography motion.

Fern Type

Organic Growth Aesthetics

  • Control mutation level, growth speed, and thickness for organic expansion trajectories.
  • Supports random palettes and fluorescent tinting for bioluminescent looks on dark backgrounds.
  • Ideal for nature, experimental, and future-ecology posters, exportable as GIF/JPG.

Popcorn Font

3D Rhythmic Volume

  • Adjust particle density, depth thickness, and blob size to build sculptural volume.
  • Control burst-growth duration and pause rotation to capture key visual moments.
  • Great for playful and event-focused key visuals with one-click export.

Dot Type

Pop Energy and Impact

  • Match typography colors to the background and tune dot density and hierarchy.
  • Supports linear connections, 3D mode switching, and explosion effects for stronger tension.
  • Suitable for pop-culture posters, exhibition graphics, and brand visual experiments.
